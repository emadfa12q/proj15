# Feature Mapping v12

## Font system

- UI is prepared to load Sahel fonts from `app/src/main/assets/fonts/`.
- Regular text: Sahel regular.
- Muted/helper text: Sahel light.
- Buttons and chips: Sahel semi-bold.
- Titles: Sahel bold.
- Main large numbers and app title: Sahel black.
- PDF report text uses the loaded Sahel regular typeface when available.

## Existing features preserved

- PIN lock and recovery.
- Realtime activity logging.
- Screenshot button instead of inline screenshot previews.
- Screenshot cleanup retention settings.
- PDF and Excel export.
- Idle/away time.
- Timeline zoom levels.
